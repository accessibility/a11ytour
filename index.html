Accessibility Tour #a11yTour 2018

a11yTour is an accessibility camp event held in both Ottawa and Montreal. This year, a11yMTL and a11yYOW joined force and organized the first accessibility camp tour. #a11ytour is a one day, one track camp, hosted consecutively in each city. We are hoping to include more cities next year

#a11ytour will start in Ottawa (Carleton university) on February 23rd, then go to Montreal (Shopify) the next day, February 24th. We will have core speakers presenting in both cities. We kept spots for "local" speakers as well.

This event is about bringing together the web accessibility community to share ideas, best practices, experiences and hopefully to raise the bar when it comes to digital inclusion. We want you at this event if you're interested in or curious about accessibility & inclusive design.
